Libraries: 
 - PyTubeFix: A bug fixed version of Pytube let user download Youtube MP3, MP4 and do some applications with Youtube link.
 - PyAutoGui: Simulate key activation

File: 

 - converter_direct: download 1 youtube song from a current opened youtube page on browser(quicker version which suppose to be binded to a hotkey)
 - converter_multiples: download all the link pasted in paste_links_here.txt(every downloaded mp3 file will be shown in the current folder and paste_links_here.txt will be wiped after application)
